Jet Engine Controller

Personal project · 2026

Bare-metal C firmware for controlling a jet engine on an RC vehicle. The architecture is state-first: a finite state machine acts as the gatekeeper for every action the system can take. This is the same design discipline used in safety-critical systems, applied here at a smaller scale.

The firmware is organized into modules: FSM core, safety layer, startup and shutdown sequences, temperature monitoring, RC input, actuator control, and display. Built with PlatformIO on a [CONFIRM: specific MCU/hardware platform]. Code and design docs are in a private repo.
